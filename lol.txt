2. USEFUL ARTISAN COMMANDS
-------------------------
# Database Commands
php artisan migrate              # Run migrations
php artisan migrate:fresh       # Drop all tables and re-run migrations
php artisan migrate:fresh --seed # Drop all tables, re-run migrations and seed
php artisan migrate:rollback    # Rollback the last migration
php artisan migrate:status      # Check migration status

# Seeder Commands
php artisan db:seed            # Run all seeders
php artisan db:seed --class=UserSeeder    # Run specific seeder
php artisan db:seed --class=ProductSeeder
php artisan db:seed --class=CategorySeeder
php artisan db:seed --class=SliderSeeder

# Cache Commands
php artisan cache:clear        # Clear application cache
php artisan config:clear       # Clear configuration cache
php artisan view:clear         # Clear view cache
php artisan route:clear        # Clear route cache

# Storage Commands
php artisan storage:link       # Create symbolic link for storage
php artisan storage:clear      # Clear storage cache

# Other Commands
php artisan serve              # Start development server
php artisan make:model -m      # Create model with migration
php artisan make:controller    # Create controller
php artisan make:seeder        # Create seeder
php artisan make:middleware    # Create middleware
php artisan make:request       # Create form request

8. TROUBLESHOOTING
-----------------
If you encounter issues:
1. Clear all caches:
   php artisan cache:clear
   php artisan config:clear
   php artisan view:clear
   php artisan route:clear

2. Check storage permissions:
   - Ensure storage directory is writable
   - Run storage:link if images don't show

3. Database issues:
   - Check .env file for correct credentials
   - Run migrations with --seed flag
   - Check database connection

4. Image upload issues:
   - Verify storage:link is created
   - Check storage permissions
   - Verify file upload size limits in php.ini
